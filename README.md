## projects

### [InDel — Parametric Income Insurance Platform](https://indel-portal.vercel.app/)
> *Go · Kafka · Zookeeper · PostgreSQL · Docker Compose · React*

Parametric income insurance for gig workers. 7-container Docker Compose stack, real-time zone-level telemetry, and a Kafka-driven claims pipeline with Razorpay UPI payouts — validated across 1,000 simulated delivery workers.

*Guidewire DEVTrails 2026 Hackathon*

---

### [Gotham — Federated Social Networking Platform](https://federated-frontend-b.vercel.app/login)
> *Go · Next.js · TypeScript · PostgreSQL · Docker*

ActivityPub-inspired federated social network with cross-server identity, messaging, and feed propagation across independently deployed nodes. Multi-service Go backend with federation, auth, and moderation; Next.js frontend with federated search and cross-instance profile resolution.

---

### [RKIVE — AI Candidate Shortlisting System](https://github.com/SudharsanSaravanan/RKIVE)
> *Python · Pandas · scikit-learn · NLP · LLMs*

Hybrid resume screening pipeline combining NER, rule-based scoring, and LLM-based resume–JD matching. Benchmarked LLM, SLM, and heuristic approaches for accuracy vs. latency tradeoffs.
